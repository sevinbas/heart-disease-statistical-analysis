Heart Disease Risk Factor Analysis

Project Overview
This project investigates the statistical relationship between clinical risk factors and heart disease presence using IBM SPSS Statistics.

Dataset
303 patients
Demographic and clinical variables
Target variable: Heart disease presence (0 = No, 1 = Yes)

Statistical Methods
Independent Samples t-test (continuous variables)
Chi-Square Test of Independence (categorical variables)
Significance level: 0.05

Key Findings
Significant differences identified in age, thalach (maximum heart rate), trestbps (resting blood pressure), and oldpeak (ST depression).
Significant associations found for sex, exercise-induced angina (exang), and number of major vessels (ca).

Tools Used
IBM SPSS Statistics
